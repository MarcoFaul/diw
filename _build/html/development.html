Development¶

Link local installation¶

  1. Install diw via Homebrew. Go here

  2. Clone & install the Repository

  3. Go to the local installed git repository directory and run the following command:

     make symlink-dev

Note

This will create a symlink between the diw binary and your local git repository

Pre-Submitting¶

Please use the make commands to format test und build local docs and check if everything works as expected
