## Xdebug

This Command enables, disables or returns the xdebug stats from the *__shop container 

``` note::
    The shell command is tuned for [Dockware](https://dockware.io/) images.
```

* Enable xdebug

         diw xdebug -e
         or
         diw xdebug --enable

* Disable xdebug

         diw xdebug
         
* Status of xdebug

         diw xdebug -s
         or
         diw xdebug --status
